
CobaltOS
========

Overview
--------

CobaltOS is A Small x64 Kernel.
Programmed In Rust, it Gaurantees Memory & Type-safety.


Features
--------

+--------------+------------+
| Architecture | Supported? |
+==============+============+
| i386         |    YES     |
+--------------+------------+
| x86-64       |    YES     |
+--------------+------------+
| powerpc      |    NO      |
+--------------+------------+
| ARM          |    NO      |
+==============+============+
